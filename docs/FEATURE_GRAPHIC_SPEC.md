# CodeVerse Feature Graphic Design Specification

## Overview
This document provides detailed specifications for creating the Google Play Store feature graphic for CodeVerse.

---

## Technical Specifications

- **Dimensions:** 1024 x 500 pixels (exact)
- **Aspect Ratio:** 2.048:1 (wide rectangle)
- **Format:** PNG or JPEG
- **Max File Size:** 15 MB
- **Color Mode:** RGB
- **Resolution:** 72 DPI (web standard)

## Typography

### Main Title: "CodeVerse" or "Learn Programming"
- **Font:** Bold Sans-serif (Arial Bold, Helvetica Bold, or similar)
- **Size:** 72-80px
- **Color:** White (`#FFFFFF`)
- **Weight:** Bold (700)
- **Letter Spacing:** 2-3px

### Subtitle: "with AI Mentor"
- **Font:** Medium Sans-serif (Arial, Helvetica)
- **Size:** 36-42px
- **Color:** Light blue (`#6C9EFF`)
- **Weight:** Medium (500)
- **Letter Spacing:** 1px

### Tagline: "Master Coding Step by Step"
- **Font:** Regular Sans-serif (Arial, Helvetica)
- **Size:** 24-28px
- **Color:** White (`#FFFFFF`) at 80% opacity
- **Weight:** Regular (400)

---

## Visual Elements

### Logo Placement
- **Position:** Left side, vertically centered
- **Size:** 200-250px width (proportional height)
- **Padding:** 40px from left edge, 40px from top/bottom
- **Style:** Use the CodeVerse logo from `assets/codeverse-logo.png`

### Background Elements (Subtle)
- **Code Brackets:** `{ }` `[ ]` `( )` - Light blue, 20% opacity
- **Semicolons:** `;` - Scattered, 15% opacity
- **Arrows:** `→` `=>` - Light blue, 15% opacity
- **Code Symbols:** `<>` `/` `*` - Very subtle, 10% opacity
- **Placement:** Scattered across background, not overlapping text

### Optional Visual Elements
- **Gradient Overlay:** Subtle dark-to-light gradient (top to bottom)
- **Grid Pattern:** Very subtle grid lines (5% opacity) for tech feel
- **Code Snippet:** Optional - faded code block in background (10% opacity)

---

## Design Style Guidelines

### Overall Aesthetic
- **Style:** Modern, clean, professional
- **Mood:** Educational, tech-forward, approachable
- **Theme:** Dark mode, programming-focused

### Design Principles
1. **Simplicity:** Don't overcrowd - less is more
2. **Readability:** Text must be readable on small screens
3. **Contrast:** High contrast between text and background
4. **Hierarchy:** Clear visual hierarchy (logo → title → subtitle → tagline)
5. **Brand Consistency:** Matches app's dark theme and color scheme

## Step-by-Step Creation Guide

### Using Canva (Recommended for Beginners)

1. **Go to Canva.com** and sign in
2. **Search:** "Google Play Feature Graphic" or create custom size (1024x500)
3. **Set Background:**
   - Color: `#0C1222` (dark blue)
   - Optional: Add subtle gradient
4. **Add Logo:**
   - Upload CodeVerse logo
   - Place on left side (40% from left)
   - Size: 200-250px width
   - Center vertically
5. **Add Text (Right Side):**
   - Main title: "CodeVerse" - White, Bold, 72px
   - Subtitle: "Learn Programming" - Light blue, 36px
   - Tagline: "with AI Mentor" - White, 24px
   - Left-align text, add spacing
6. **Add Background Elements:**
   - Add code symbols (brackets, semicolons)
   - Set opacity to 10-20%
   - Scatter across background
7. **Export:**
   - Format: PNG
   - Size: 1024x500 pixels
   - Quality: High

### Using Figma (For Designers)

1. **Create Frame:** 1024 x 500px
2. **Background:** Rectangle, fill `#0C1222`
3. **Gradient Overlay:** Optional, subtle
4. **Logo:** Import CodeVerse logo, position left
5. **Text Layers:**
   - Title: "CodeVerse" - White, Bold, 72px
   - Subtitle: "Learn Programming" - `#6C9EFF`, 36px
   - Tagline: "with AI Mentor" - White 80%, 24px
6. **Background Symbols:** Add code symbols, low opacity
7. **Export:** PNG, 1024x500
