# Feature Graphic Creation Prompts

Use these prompts to generate your CodeVerse feature graphic with AI image generators or provide to designers.

---

## Prompt for AI Image Generators (DALL-E, Midjourney, Stable Diffusion, etc.)

### Full Prompt (Recommended)
```
Create a Google Play Store feature graphic banner, 1024x500 pixels, horizontal layout. 
Design: Modern, clean, professional dark theme banner for a programming education app.

Left side (40%): Large CodeVerse logo, white/light blue, centered vertically. 
Right side (60%): Bold white text "CodeVerse" (72px, bold) at top, light blue text "Learn Programming" (36px) below it, smaller white text "with AI Mentor" (24px) at bottom. All text left-aligned with generous spacing.

Background: Dark blue (#0C1222) with subtle gradient to slightly lighter blue (#1A2332). 
Subtle programming code symbols scattered in background at 10-20% opacity: curly brackets { }, square brackets [ ], parentheses ( ), semicolons ;, arrows →, code symbols < > / *.

Style: Minimalist, modern, tech-forward, educational. High contrast, readable text. 
No clutter, clean design. Dark mode aesthetic matching programming/developer tools.

Colors: Background #0C1222 (dark blue), text white #FFFFFF and light blue #6C9EFF, 
subtle code symbols in light blue at low opacity.

Professional, trustworthy, educational programming app banner suitable for Google Play Store.
```

## Detailed Prompt for Advanced AI Generators

```
Create a professional Google Play Store feature graphic banner:

SPECIFICATIONS:
- Dimensions: 1024 pixels wide x 500 pixels tall
- Format: Horizontal rectangle banner
- Style: Modern, minimalist, professional, tech-forward

LAYOUT:
- Left section (40% width): CodeVerse app logo, large, white/light blue color, 
  vertically centered, 200-250px width, 40px padding from edges
- Right section (60% width): Text content, left-aligned, vertically centered
  - Main title: "CodeVerse" - White (#FFFFFF), Bold, 72-80px font size
  - Subtitle: "Learn Programming" - Light blue (#6C9EFF), Medium weight, 36-42px
  - Tagline: "with AI Mentor" - White at 80% opacity, Regular weight, 24-28px
  - Generous spacing between text elements

BACKGROUND:
- Base color: Dark blue (#0C1222)
- Subtle gradient: Dark blue (#0C1222) to slightly lighter (#1A2332), top to bottom
- Decorative elements: Very subtle programming code symbols scattered:
  - Curly brackets { }, square brackets [ ], parentheses ( )
  - Semicolons ;, arrows →, code symbols < > / *
  - All symbols in light blue (#6C9EFF) at 10-20% opacity
  - Randomly scattered, not overlapping text

DESIGN PRINCIPLES:
- Clean, uncluttered design
- High contrast between text and background
- Readable on small screens (mobile-first)
- Professional, trustworthy appearance
- Educational, approachable tone
- Matches dark mode programming app aesthetic

COLOR PALETTE:
- Background: #0C1222 (dark blue)
- Background gradient: #1A2332 (lighter blue)
- Primary text: #FFFFFF (white)
- Secondary text: #6C9EFF (light blue)
- Accent: #3B82F6 (blue)

TEXT HIERARCHY:
- Logo: Prominent, clear, recognizable
- Title: Largest, boldest, most visible
- Subtitle: Medium size, secondary importance
- Tagline: Smallest, supporting information

STYLE NOTES:
- No pixelation or blurriness
- Sharp, crisp edges
- Professional quality suitable for app store
- Modern, contemporary design
- Educational technology aesthetic
- Programming/developer tools inspired

Avoid: Cluttered design, low contrast, small unreadable text, bright colors, 
excessive decorative elements, overlapping text and logo.
```

## Prompt for Canva/Design Tools

### Step-by-Step Instructions
```
1. Create a custom design: 1024 x 500 pixels

2. Background:
   - Fill with color #0C1222 (dark blue)
   - Optional: Add subtle gradient overlay (dark to slightly lighter blue)

3. Add Logo (Left Side):
   - Upload CodeVerse logo
   - Position: Left side, 40% from left edge
   - Size: 200-250px width
   - Center vertically
   - Color: White or light blue

4. Add Text (Right Side):
   - Text 1: "CodeVerse" - White, Bold, 72px, left-aligned
   - Text 2: "Learn Programming" - Light blue (#6C9EFF), Medium, 36px
   - Text 3: "with AI Mentor" - White, Regular, 24px
   - Position: Right side, 60% width, left-aligned, vertically centered
   - Add spacing between text elements

5. Add Decorative Elements:
   - Add code symbols: { } [ ] ( ) ; → < > / *
   - Color: Light blue (#6C9EFF)
   - Opacity: 10-20%
   - Scatter across background
   - Don't overlap text or logo

6. Export:
   - Format: PNG
   - Size: 1024x500 pixels
   - Quality: High
```
